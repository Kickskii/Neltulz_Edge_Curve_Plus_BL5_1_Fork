Key Features

  • Insert single edge loop with flow
  • Insert multiple edge loops with flow
  • Control the tension of the edge loops
  • Control the number of iterations (more iterations = better result)
  • Control the minAngle of the edge loops. Higher numbers can fix problems like, bulging.
  • Modern Overhaul: Fully rebuilt for Blender 5.1 using BMesh for high stability and speed.
  • Native UI: Integrated with Blender's native sub-panels and split layouts.

Dependencies

Neltulz - Edge Curve requires the EdgeFlow addon for blender to be installed prior to using. Download the latest version here: EdgeFlow Addon for Blender

Installation

  1. Download the Neltulz - Edge Curve zip file anywhere on your computer.
  2. Run Blender.
  3. Go to Edit > Preferences.
  4. In the left sidebar, go to Add-ons.
  5. At the top right, click Install.
  6. Browse for the zip file and click Install Add-on.
  7. Enable the checkbox next to Neltulz - Edge Curve.

How to Use

  1. Select an object and switch to Edit Mode.
  2. Select one or more Edge(s).
  3. Press CTRL+SHIFT+E to insert an edge loop with flow.
  4. Configure the options in the sidebar or the redo panel!
Hotkeys
Hotkey Result Command Properties
ALT+SHIFT+E Insert Edge Loop(s) with Flow ntzedgcrv.insertedges numSegments = 1
